# 『封域異獣録』技能・状態設定

版：0.1  
更新日：2026年7月13日

## 1. 方針

- 縦切り版の正式技能は8個に限定する
- 直接ダメージは種ごとの基礎行動で確保する
- 正式技能は回復、鎮静、防御、状態相互作用を中心にする
- 技能の数値はコードへ埋め込まずマスターデータで調整する
- 状態付与を確率抽選にせず、条件と持続時間を予測表示する

## 2. 基礎行動

基礎行動は活性値を20回復する。正式な継承技能には含めず、種または個体の基本身体構造に結び付ける。

| 種 | 基礎行動 | 区分 | 威力 | 追加効果 |
|---|---|---|---:|---|
| トモシゴケ | 灯角突き | 物理 | 14 | なし |
| ヌマクグリ | 大尾打ち | 物理 | 16 | なし |
| スミワタリ | 水圧弾 | 生態 | 14 | なし |
| キリハネ | 翅刃 | 生態 | 15 | 霧中なら威力17 |
| レキマトイ | 殻ばさみ | 物理 | 18 | なし |

ニゴリグイのボス行動は別途定義する。

## 3. 縦切り版の8技能

### 3.1 灯苔の雫

| 項目 | 内容 |
|---|---|
| ID | `moss_droplet` |
| 主な習得種 | トモシゴケ |
| 対象 | 味方単体 |
| 活性消費 | 25 |
| 回復力 | 18 |
| 区分 | 回復 |

```text
回復量
＝ floor(18 × (1 ＋ 使用者の感応 ÷ 100))
```

対象が汚染状態の場合でも回復はするが、汚染自体は解除しない。

### 3.2 静かな明滅

| 項目 | 内容 |
|---|---|
| ID | `calming_glimmer` |
| 主な習得種 | トモシゴケ |
| 対象 | 敵単体 |
| 活性消費 | 20 |
| 区分 | 鎮静・調査支援 |

効果：

- 対象へ鎮静を付与する
- 野生個体の警戒度を20下げる
- すでに鎮静中の場合、警戒度低下だけを適用する
- 鎮静の持続時間は感応と抵抗の差で決める

### 3.3 身代わり潜行

| 項目 | 内容 |
|---|---|
| ID | `burrow_guard` |
| 主な習得種 | ヌマクグリ |
| 対象 | 自分以外の味方単体 |
| 活性消費 | 20 |
| 優先度 | 高 |
| 区分 | 防御 |

効果：

- ラウンド終了まで対象への単体攻撃を使用者が引き受ける
- 引き受けた被害を最終計算後80%に軽減する
- 全体攻撃と環境被害は引き受けない
- 使用者が行動不能または戦闘不能になった時点で解除する

### 3.4 泥幕

| 項目 | 内容 |
|---|---|
| ID | `mud_screen` |
| 主な習得種 | ヌマクグリ |
| 対象 | 敵全体 |
| 活性消費 | 25 |
| 区分 | 環境・弱体 |

効果：

- 対象を湿潤状態にする
- 対象の機敏を20%下げる
- 環境由来の効果として基本2ラウンド固定
- すでに湿潤の場合は持続時間を更新する

### 3.5 澄み流し

| 項目 | 内容 |
|---|---|
| ID | `clarifying_flow` |
| 主な習得種 | スミワタリ |
| 対象 | 味方単体 |
| 活性消費 | 25 |
| 回復力 | 8 |
| 区分 | 浄化・回復 |

効果：

- 汚染を優先して、解除可能な不利状態を1つ解除する
- 解除後、対象を小回復する
- 解除対象がなくても小回復は発生する

### 3.6 霧包み

| 項目 | 内容 |
|---|---|
| ID | `mist_shroud` |
| 主な習得種 | キリハネ |
| 対象 | 戦場全体 |
| 活性消費 | 25 |
| 区分 | 環境・弱体 |

効果：

- 戦場を霧中状態にする
- 敵側の攻勢を15%下げる
- 基本2ラウンド固定
- キリハネの基礎行動「翅刃」の威力を15から17へ変更する
- 霧中を前提とする観察条件を満たせる

### 3.7 礫砕き

| 項目 | 内容 |
|---|---|
| ID | `rubble_breaker` |
| 主な習得種 | レキマトイ |
| 対象 | 敵単体 |
| 活性消費 | 30 |
| 威力 | 24 |
| 区分 | 物理攻撃・装甲対策 |

効果：

- 物理ダメージを与える
- 対象が装甲状態なら装甲を解除する
- 装甲を解除した場合、対象へ露出を1ラウンド付与する
- 対象が装甲状態でなくても通常ダメージは発生する

### 3.8 纏い殻

| 項目 | 内容 |
|---|---|
| ID | `mantled_shell` |
| 主な習得種 | レキマトイ |
| 対象 | 自分 |
| 活性消費 | 20 |
| 区分 | 防御強化 |

効果：

- 自分へ装甲を付与する
- 装甲中は受ける物理ダメージを最終計算後75%にする
- 基本2ラウンド固定
- 防御コマンドと併用する場合、軽減は乗算する

## 4. 状態一覧

| 状態 | 対象 | 効果 | 基本持続 |
|---|---|---|---:|
| 湿潤 | 個体 | 霧・泥技能の条件。機敏低下等は発生源で定義 | 2 |
| 霧中 | 戦場 | 敵攻勢低下、キリハネ技能強化 | 2 |
| 汚染 | 個体 | ラウンド終了時に生態ダメージ | 2 |
| 装甲 | 個体 | 物理ダメージを75%に軽減 | 2 |
| 露出 | 個体 | 受ける物理ダメージを125%に増加 | 1 |
| 鎮静 | 野生個体 | 攻勢15%低下、警戒度上昇行動を抑制 | 1〜3 |

### 汚染ダメージ

縦切り版の初期案では、最大HPの5%をラウンド終了時に失う。最低1、端数切り上げ。ボスギミックでは個別の汚染量を設定できる。

### 軽減の重複

防御中かつ装甲状態の場合：

```text
最終ダメージ
＝ 通常ダメージ × 0.60 × 0.75
＝ 通常ダメージ × 0.45
```

身代わり潜行まで重なる場合は、最後に0.80を掛ける。

## 5. 継承候補

8技能はすべてデータ上に `inheritanceCost` を持つ。初期案は次のとおり。

| 技能 | 継承容量 |
|---|---:|
| 灯苔の雫 | 3 |
| 静かな明滅 | 3 |
| 身代わり潜行 | 4 |
| 泥幕 | 3 |
| 澄み流し | 4 |
| 霧包み | 3 |
| 礫砕き | 4 |
| 纏い殻 | 4 |

技能の継承可否と容量はプレイテストで変更できる。
