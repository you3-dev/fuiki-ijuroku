# 『封域異獣録』異獣設定

版：0.1  
更新日：2026年7月13日

## 1. 共通方針

- 名称は研究者や地域住民が外見・習性から付けた和名とする
- セーブデータとマスターデータでは変更されない英字IDを使う
- かわいさだけ、恐ろしさだけに偏らず、生物としての説得力を持たせる
- 生態が戦闘技能、協力条件、共鳴継承へつながるようにする
- 属性だけで個体を分類せず、系統、環境適応、生態タグを分けて持つ

## 2. トモシゴケ

| 項目 | 内容 |
|---|---|
| ID | `tomoshigoke` |
| 区分 | 初期同行・仲間化可能 |
| 系統候補 | 植生／小獣 |
| 環境タグ | 湿潤、微光、共生 |
| 戦闘役割 | 回復、支援、警戒度低下 |

### 外見

イタチと小型の鹿を合わせたような四足獣。背中から尾にかけて柔らかな苔が生え、頭部には短い枝角状の器官がある。

苔の内部には淡い青緑色の生命核組織が走り、健康な水辺では呼吸に合わせて穏やかに発光する。汚染された場所では光が弱まり、苔の先が灰色になる。

### 生態

背中の苔は別の植物ではなく、トモシゴケの皮膚器官と一体化した共生組織。水中の微細な魔力を蓄え、夜間の体温維持と傷の修復に使う。

他の小型異獣が発光を目印に集まるため、湿原では「歩く灯台」と呼ばれる。争いを避ける性質があり、仲間が興奮すると低い音と光の明滅で落ち着かせる。

### 戦闘

- 味方単体の小回復
- 味方全体の弱い継続回復
- 野生個体の警戒度を下げる
- 観察段階を補助する
- 攻撃能力と耐久は低め

### 固有特性候補

**灯苔の呼吸**：ラウンド終了時、最もHP割合の低い味方をわずかに回復する。

### 図鑑上の見せ場

水質の変化に応じて苔の色が変わるため、灰苔湿原の環境状態を視覚的に示す指標になる。

## 3. ヌマクグリ

| 項目 | 内容 |
|---|---|
| ID | `numakuguri` |
| 区分 | 初期同行・仲間化可能 |
| 系統候補 | 湿生／両生 |
| 環境タグ | 泥地、潜行、群れ |
| 戦闘役割 | 防御、かばう、命中低下 |

### 外見

サンショウウオとカワウソを合わせたような低い体形。胴体より長い平たい尾を持ち、前脚は短く、後脚は泥を押し出すために太い。

皮膚は焦げ茶から鈍い青灰色。頭部の左右に泥中の振動を読むひげ状器官がある。

### 生態

泥の中へ潜り、尾で水と土を混ぜて柔らかな水路を作る。小型異獣が乾季を越すための避難穴として利用することから、湿原の「土木屋」とも呼ばれる。

群れの仲間が危険にさらされると、自分の尾で泥幕を作って視界を遮る。

### 戦闘

- 味方単体をかばう
- 自分の物理被害を軽減する
- 泥幕で敵の命中を下げる
- 行動は遅めだが耐久が高い

### 固有特性候補

**泥の外套**：各戦闘で最初に受ける物理攻撃の被害を軽減する。

### 図鑑上の見せ場

ヌマクグリが作った水路を調べることで、通常とは異なる調査地点や採取場所を発見できる。

## 4. スミワタリ

| 項目 | 内容 |
|---|---|
| ID | `sumiwatari` |
| 区分 | 野生・仲間化可能 |
| 系統候補 | 水生／軟体 |
| 環境タグ | 濾過、透明、汚染耐性 |
| 戦闘役割 | 状態異常解除、浄化、回復補助 |

### 外見

ウミウシと半透明のリボンを合わせたような軟体種。水中では長い身体を波打たせて泳ぎ、浅瀬では腹面を広げて滑る。

体内に幾層もの濾過膜が見え、取り込んだ汚れが灰色の筋として移動する。正常時は水晶のように透明だが、汚染が蓄積すると黒く濁る。

### 生態

大量の水を体内へ通し、微生物、鉱物片、過剰な魔力を分離する。濾過した物質の一部は生命核の周囲へ一時保管され、休息時に安全な塊として排出される。

浄化能力を超える汚染を取り込むと自力で排出できず、興奮して周囲へ腐食性の液体を撒く。

### 協力条件

1. 体内に溜まった黒い沈殿物を観察する。
2. 汚染除去具を使用して濾過器官を助ける。
3. 警戒度が下がった個体へ協力要請する。

### 戦闘

- 毒、汚染、能力低下を解除する
- 味方へ弱い耐性を付与する
- 敵の強化を一部洗い流す
- 攻撃能力は低め

### 固有特性候補

**澄化膜**：自分が状態異常を解除した時、対象をわずかに回復する。

## 5. キリハネ

| 項目 | 内容 |
|---|---|
| ID | `kirihane` |
| 区分 | 野生・仲間化可能 |
| 系統候補 | 翅虫／飛翔 |
| 環境タグ | 霧、鱗粉、音響 |
| 戦闘役割 | 素早さ、回避、敵弱体化 |

### 外見

蛾と小鳥を合わせたような小型飛翔種。四枚の翅は薄い膜状で、表面に細かな鱗粉生成器官が並ぶ。尾は短く、空気の流れを読む房状器官になっている。

翅は青灰色を基調とし、生命核に近い部分だけ鈍い紫に光る。

### 生態

空気中の水分を翅の表面へ集め、鱗粉を核にして局所的な霧を作る。霧は捕食者から身を隠すだけでなく、仲間同士の音を遠くまで伝える役割を持つ。

霧の中で不用意に動く相手を脅威と判断し、さらに濃い霧で方向感覚を奪う。

### 協力条件

1. キリハネが霧を発生させたラウンドは攻撃せず、防御または観察する。
2. 霧の中で発する鳴き声を観察する。
3. 鎮静音具で同じ周期の音を返す。

### 戦闘

- 高い素早さ
- 味方の回避または行動順を上げる
- 敵の命中を下げる
- 耐久は低め

### 固有特性候補

**霧渡り**：霧または湿潤環境では素早さが上がる。

## 6. レキマトイ

| 項目 | 内容 |
|---|---|
| ID | `rekimatoi` |
| 区分 | 野生・仲間化可能 |
| 系統候補 | 鉱殻／甲殻 |
| 環境タグ | 遺跡、装甲、共鳴感知 |
| 戦闘役割 | 物理攻撃、防御、装甲破壊 |

### 外見

蟹とアルマジロを合わせたような六脚の甲殻種。成長するたびに周辺の石や遺跡片を選び、粘着性の分泌物で背中へ固定する。

自然石だけをまとう個体は丸みを帯びるが、遺跡片をまとう個体には人工的な直線や発光紋が現れる。

### 生態

外殻の材料として硬い物を集めるが、強度だけでなく生命核と共鳴する振動を選んでいる。遺跡装置が暴走すると不安定な破片まで取り込み、痛みと過剰な振動で攻撃的になる。

### 協力条件

1. 周囲の遺跡波動または発生装置を停止する。
2. 外殻を破壊せず、防御や弱い攻撃で警戒度を下げる。
3. 安定した遺跡片を調査道具として与える。

### 戦闘

- 高い物理攻撃と防御
- 自分の装甲を強化する
- 敵の防御を削る
- 素早さと感応系の能力は低め

### 固有特性候補

**纏い殻**：防御上昇を受けた時、次の物理攻撃もわずかに強くなる。

## 7. ニゴリグイ

| 項目 | 内容 |
|---|---|
| ID | `nigorigui` |
| 区分 | 地域ボス・縦切り版では通常加入不可 |
| 系統候補 | 浄環／大型湿生 |
| 環境タグ | 浄化施設、汚染吸収、古代共生 |
| 戦闘役割 | 汚染蓄積、全体攻撃、環境操作 |

### 外見

巨大なサンショウウオと濾過槽を組み合わせたような大型異獣。背中から複数の濾過筒が伸び、その一部は古代施設の管路と生体的に接続している。

腹部の生命核は通常なら淡い水色だが、縦切り版開始時には汚染物を蓄積して黒紫色に脈動する。

### 生態

古代施設が稼働していた時代から中枢槽に生息し、水路の沈殿物を摂取してきた。施設はニゴリグイへ安全な排出経路を提供し、ニゴリグイは濾過膜では処理できない物質を分解する共生関係にある。

補助装置の故障で排出経路が閉ざされ、体内の汚染濃度が限界を超えた結果、周囲を敵と認識して暴走している。

### ボス戦

1. 体表に付着した汚染塊を除去する。
2. 暴走している補助装置を停止する。
3. 大量排出行動を防御して耐える。
4. 生命核の過負荷状態を観察する。
5. 浄化経路を復旧し、鎮静を成立させる。

HPをゼロにすることは失敗または不完全な解決として扱う。正規の勝利条件は環境ギミックの解決と鎮静である。

### 固有特性候補

**濁り喰い**：状態異常や汚染を吸収し、自分の行動を強化する。ただし蓄積しすぎると過負荷になる。

## 8. 共鳴継承の縦切り版候補

種の組み合わせはデータ検証用の暫定案。最終結果は技能・能力設計後に調整する。

| 基体 | 共鳴相手 | 触媒 | 結果候補 |
|---|---|---|---|
| トモシゴケ | ヌマクグリ | なし | トモシゴケ（湿生傾向） |
| ヌマクグリ | トモシゴケ | なし | ヌマクグリ（共生傾向） |
| スミワタリ | キリハネ | 霧触媒 | スミワタリ（霧適応） |
| レキマトイ | スミワタリ | 浄化触媒 | レキマトイ（澄殻傾向） |
| トモシゴケ | レキマトイ | 遺跡触媒 | 縦切り版の特殊分岐候補 |

特殊分岐を6種の外へ増やすか、既存種の適応差分に留めるかは、グラフィック工数を確認して決定する。
