# 浄化施設中枢・地域ボス実装記録

## 1. 実装の目的

灰苔湿原の縦切りを、浄化施設中枢の地域ボス「ニゴリグイ」から地域報告まで完結させた。単純なHP削りではなく、施設の故障と異獣の生態を同時に読み解き、ニゴリグイを倒さず浄化行動へ戻す調査戦とした。

成功時の到達状態は次のとおり。

- 先遣隊記録 3/3
- 水路弁復旧 2/2
- 異獣録 6/6
- 灰苔湿原の地域調査完了

## 2. 3段階の調査戦

### 第1段階：左右の汚染塊を除去

最初に排出口を観察し、レキマトイの礫砕きで左右の装甲を外す。スミワタリの「澄み流し」で露出した汚染塊を除去すると、ニゴリグイ本体の制御部へ進める。

### 第2段階：排出暴発へ備える

制御盤を解析すると、次に排出路を開いたラウンドで全体攻撃が起きることが分かる。前衛3体が防御した状態で中央排出路を開くと、暴発を安全な排出へ変え、過負荷を抑えられる。

### 第3段階：過負荷と警戒を解消

スミワタリの「澄み流し」で過負荷を、トモシゴケの「静かな明滅」で警戒度を下げる。過負荷20以下、警戒度20以下、排出路2/2、鎮静済みを満たすと、主人公が浄化経路を接続できる。

推奨手順では9ラウンドで成功する。途中の乱数はseedで再現され、各ラウンドは行動確定と解決の2段階で保存される。

## 3. 失敗と再試行

次の場合は調査失敗になる。

- ニゴリグイ本体の生態HPを0にしてしまう
- 前衛3体が全滅する
- 施設の過負荷を許容値以上へ悪化させる

失敗時は地域全体をやり直さず、中枢到達直後の初期状態から再試行する。解決済みラウンドを再度解決しても状態が二重に進まない。

## 4. 地域報告

浄化成功後、第3記録「通常回線へ、この記録を流すな」を復元する。プレイヤーは報告の強調点を次の3つから選ぶ。

1. 異獣との共同作業
2. 施設の危険性
3. 外部からの制御痕跡

選択は研究所の地域完了報告へ反映する。報告を確定すると第3記録と研究更新を一度だけ付与し、同じ地域へ戻っても重複しない。

## 5. 保存と入力検査

セーブスキーマを版6へ更新し、版1〜版5からの移行で次を補完する。

- 中枢ボス戦の状態
- 地域報告の選択
- 地域完了フラグ

入力検査では、段階、左右の汚染塊、装甲、排出路、過負荷、警戒度、鎮静、勝敗、画面フェーズの整合を確認する。地域完了時には濾過樹群完了、報告選択、先遣隊記録3/3が必要である。

## 7. 検証結果

| 確認項目 | 結果 |
|---|---|
| 推奨9ラウンドで非討伐成功 | 単体テスト、統合テスト、ブラウザで確認 |
| 排出暴発の予告と3体防御 | 単体テスト、ブラウザで確認 |
| 本体HP0、前衛全滅、過負荷の失敗 | 単体テストで確認 |
| 失敗後の再試行と二重解決防止 | 単体テストで確認 |
| 版5セーブの版6移行 | バックアップテスト、ブラウザで確認 |
| 地域報告と報酬の一度限りの反映 | 統合テスト、ブラウザで確認 |
| Lint、型検査、Vitest 49件、本番ビルド | 成功 |

ブラウザでは、版5から移行した既存セーブを使い、中枢開始、第1〜第3段階、報告方針「異獣との共同作業」、地域完了までを実際の画面で通した。最終画面で記録3/3・弁2/2・異獣録6/6を確認した。

## 8. 次の工程

まず灰苔湿原を入口から地域完了まで通してプレイし、次を記録する。

- 3体への指示量に見合う判断があるか
- ボスの次の操作が画面だけで理解できるか
- 地点選択でも未知を調べる感覚があるか
- 30〜60分の縦切り目標に収まるか

その後、未実装の個体育成と非消失型「共鳴継承」を最小構成で接続する。探索評価が低い場合だけ、灰苔湿原のデータを再利用してタイルマップ自由移動の比較試作を行う。
