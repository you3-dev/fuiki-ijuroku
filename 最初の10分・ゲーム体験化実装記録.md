# 『封域異獣録』最初の10分・ゲーム体験化実装記録

更新日：2026年7月15日  
対象：`app/`  
判定：タイトルから初回加入まで、説明なしで進められるゲーム導線として成立

## 1. 改修の目的

実機確認で判明した「操作が分からない」「文字中心でゲーム感がない」「何が起きているか分からない」という問題に対し、最初の10分を次の体験へ再構成した。

```text
タイトル
→ 3枚の短いプロローグ
→ 最初の任務と同行個体を確認
→ 湿原入口を観察
→ 浅瀬で初回接触
→ 観察・防御・浄化・鎮静・協力
→ スミワタリ加入と技能解放
```

## 2. 実装した内容

### 導入

- 専用タイトル画面と「調査を始める／記録から再開」の分岐
- 世界、異常、主人公の任務を1文ずつ伝える3枚のプロローグ
- 初回だけ情報量を抑えた「出発前ブリーフィング」
- 常に一つだけ表示する「次にすること」案内
- 調査開始後の目的表示と、入口から浅瀬までの簡易経路図

### 初回観察戦

- スミワタリの汚染時・浄化後イラスト
- 5段階の協力条件と現在位置の表示
- 各ラウンドで推奨行動を一つだけ強調
- 行動の意味を「原因を見る」「時間を作る」「詰まりを除く」「敵意がないと伝える」と物語上の因果で説明
- 加入時に前衛3体の完成と新技能「澄み流し」を演出

### ビジュアル

次のオリジナル画像を内蔵画像生成で作成し、WebPへ最適化した。

- `app/public/art/title-graymoss.webp`：灰苔湿原と古代錬成施設
- `app/public/art/sumiwatari-encounter.webp`：汚染され苦しむスミワタリ
- `app/public/art/sumiwatari-cleansed.webp`：浄化後の同一個体

画像は生態図鑑と抑制したゲームコンセプトアートの中間を狙い、既存作品の固有意匠やキャラクターは使用していない。今後の正式アート制作でも、この色調と自然誌的な描写を基準にできる。

## 4. 2人体制レビューで反映した方針

- 30秒以内に「異獣の異常調査と先遣隊記録の回収」が分かる導入
- 研究設備など、最初の任務に不要な情報は初加入まで非表示
- 初戦は自由度より因果関係の理解を優先
- セーブスキーマを変更せず、既存データをそのまま継続可能
- 320px幅、動きを減らす設定、再読込復帰を受け入れ条件として維持

## 5. 現時点での評価

改修前は完成済みのシステムを読む試作品だった。改修後は、世界へ入り、異獣を助け、仲間が増え、技能が解放されるところまでを一続きの報酬として体験できる。ゲームの中心的な楽しさは「倒す」のではなく、「観察によって苦しみの原因を知り、適切な処置で協力関係を作る」部分として見えるようになった。

次の優先候補は、効果音と短い環境音、既存6体の正式図版、加入後の最初の自由選択に対する報酬強化である。タイルマップ自由移動との比較は、この地点選択版を実機で再評価してから行う。
