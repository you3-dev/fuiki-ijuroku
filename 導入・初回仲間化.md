# 『封域異獣録』導入・初回仲間化設計

版：0.1  
更新日：2026年7月13日  
対象：縦切り版の開始から前衛3体完成まで

## 1. 採用方針

前衛は「最大3体」とし、空き枠を許可する。主人公はトモシゴケとヌマクグリの2体で最初の調査へ出発し、灰苔の浅瀬でスミワタリへの観察・鎮静・協力要請を成功させて前衛3体を完成させる。

貸与個体や7種目の専用異獣は追加しない。

## 2. この導入で教えること

1. 前衛は最大3体で、いる個体にだけ指示を出す
2. 主人公の観察は攻撃とは別の調査行動である
3. 異獣の危険行動には生態上の原因がある
4. 防御や鎮静で戦闘を解決できる
5. 条件達成後の協力要請は確定成功する
6. 加入した個体で前衛の役割が増える

最初から3体分の技能を説明せず、2体で基本操作を覚えた直後に3体目を加える。

## 3. 編成ルール

| 項目 | ルール |
|---|---|
| 調査隊 | 最大6体 |
| 前衛 | 1〜3体 |
| 控え | 0〜3体 |
| 調査開始 | 行動可能な前衛が1体以上必要 |
| 推奨編成 | 前衛3体。少ない場合は警告するが禁止しない |
| 行動入力 | 現在行動可能な前衛にだけ指示する |
| 敗北 | 出撃中の前衛と控えがすべて戦闘不能 |

初回調査ではトモシゴケとヌマクグリの2体を必須とする。以後は1〜3体でも出発できるが、少数編成は任意の挑戦として扱う。

## 4. 初回プレイフロー

```text
主人公設定
  ↓
研究所でトモシゴケとヌマクグリを受け取る
  ↓
2体編成で灰苔湿原へ出発
  ↓
湿原入口で地点選択と保存を学ぶ
  ↓
灰苔の浅瀬で汚染されたスミワタリと遭遇
  ↓
観察 → 汚染除去 → 鎮静 → 協力要請
  ↓
スミワタリ加入・呼称設定
  ↓
前衛3体の入力確認
  ↓
観測櫓跡／沈み水路の分岐を解放
```

## 5. 灰苔の浅瀬：初回遭遇

### 状況

浅瀬の水が急に黒く濁り、半透明の異獣が苦しむように水を跳ね上げる。スミワタリは主人公たちを追い払おうとするが、攻撃行動と同時に濾過膜の一部を何度も地面へ擦りつける。

主人公は、単なる敵意ではなく濾過器官の閉塞が原因だと仮説を立てる。

### 初期状態

| 項目 | 値・状態 |
|---|---|
| スミワタリの警戒度 | 60 |
| 生態状態 | 汚染 |
| 協力要請 | 条件未確認のため選択不可 |
| 味方 | トモシゴケ、ヌマクグリ |
| 控え | なし |
| 支給品 | 汚染除去具1個。初回イベント専用で消費損にならない |

## 6. チュートリアル手順

### 第1段階：観察

主人公の調査支援「観察」をスミワタリへ使う。

判明する情報は次のとおり。

- 黒い濁りは体表ではなく濾過膜の内側にある
- 攻撃の直前に詰まった器官を擦っている
- 汚染を除けば警戒が下がる可能性が高い

観察後、協力条件の1段階目を異獣録へ記録する。

### 第2段階：身を守る

スミワタリは「濁り水圧」を予告する。ヌマクグリの防御または身代わり行動を案内し、攻撃しなくてもラウンドを進められることを教える。

選択を強制しすぎず、別行動を選んでも敗北しない威力にする。

### 第3段階：汚染除去

主人公が支給品「汚染除去具」を使用する。

- 汚染状態を解除
- 警戒度を20下げる
- スミワタリの外見を黒い濁りから半透明へ変える
- 協力条件の2段階目を記録

道具は主人公の調査支援枠を使い、味方2体は同じラウンドに行動できる。

### 第4段階：鎮静

トモシゴケの「静かな明滅」を使う。

- 警戒度を20下げる
- 鎮静状態を付与
- 協力要請を有効化

技能を使わず戦闘を続けてもよいが、次ラウンドに短いヒントを表示する。初回だけ活性値不足にならないよう開始値を保証する。

### 第5段階：協力要請

次の条件をすべて満たした状態で、主人公が協力要請を行う。

```text
観察済み
＋ 汚染解除済み
＋ 警戒度20以下
＋ 鎮静状態
＝ 協力要請成功
```

成功率の表示や抽選は行わない。条件を満たした時点で「協力を求められます」と明示する。

## 7. 失敗させない仕組み

- スミワタリを攻撃しても初回は倒れる直前で「弱っている」と表示し、致命打を保留する
- 汚染除去具を誤対象へ使えないようにする
- 活性値不足で静かな明滅を使えない状態にしない
- 主人公の観察・道具・協力要請は同じラウンドに1つだけ行う
- 同じ段階で2ラウンド迷った場合は、観察記録としてヒントを出す
- パーティが全滅した場合は浅瀬の遭遇直前へ戻し、道具を復元する

チュートリアルは手順を教えるが、正解ボタンだけを順番に光らせる操作にはしない。観察結果を読み、自分で次の行動を選べる余地を残す。

## 8. 加入処理

協力要請成功後、戦闘を終了して次を一括確定する。

1. スミワタリの個体を生成する
2. 初期前衛の空き枠へ配置する
3. 異獣録を「初観察」へ更新する
4. 協力条件を既知情報として記録する
5. 汚染除去具のチュートリアル所持状態を終了する
6. 初回仲間化フラグを立てる
7. 重要保存を完了する
8. 呼称入力と前衛3体の説明へ進む

加入個体は戦闘後加入のため、その遭遇の経験値を得ない。提供元や貸与個体ではなく、以後も所有個体として残る。

## 9. 加入個体の初期状態

| 項目 | 初期値 |
|---|---|
| 種 | スミワタリ |
| レベル | 初期同行2体より1低い値を候補とする |
| 信頼度 | 1 |
| 成長傾向 | 初回用の均整型。完全ランダムにしない |
| 初期技能 | 水圧弾、澄み流し |
| 状態 | 汚染解除済み、HPは安全圏まで回復 |
| 配置 | 前衛3枠目 |

呼称は初期候補を提示し、入力を省略して後から変更できるようにする。

## 10. 3体編成の説明

加入後、次の通常遭遇前に短い編成説明を行う。

- 前衛札が2枚から3枚へ増える
- 各ラウンドで3体分の行動を入力する
- トモシゴケは回復と鎮静
- ヌマクグリは防御とかばう
- スミワタリは浄化と状態解除
- 「前回を再現」「全員基礎」は3体入力後に紹介する

同時に全コマンドを説明せず、次の汚染状態を含む遭遇でスミワタリの役割を実践させる。

## 11. 再調査時

初回仲間化後、灰苔の浅瀬では通常のスミワタリまたは汚染個体が出現する。

- 既知の協力条件を戦闘画面に表示する
- 汚染除去具以外の浄化手段でも条件を満たせる
- 仲間化するか戦闘を終えるかは任意
- 初回の致命打保留は適用しない
- 条件達成後の協力要請は引き続き確定成功

## 12. セーブ境界

| 地点 | 保存内容 |
|---|---|
| 灰苔の浅瀬へ入る前 | 遭遇前チェックポイント |
| 各ラウンド解決後 | 戦闘状態、観察段階、警戒度、道具状態 |
| 協力要請成功時 | 個体追加、前衛配置、図鑑更新を一括保存 |
| 呼称入力後 | 呼称のみ通常保存 |

再読込でスミワタリが二重加入しないよう、個体追加と `firstRecruitmentCompleted` を同じトランザクションで保存する。

## 13. テスト項目

- 前衛2体で調査と戦闘を開始できる
- 存在しない3枠目の行動入力を要求しない
- 観察前に協力要請できない
- 汚染除去で警戒度と外見が更新される
- 静かな明滅で協力可能状態になる
- 条件達成後の協力要請が必ず成功する
- 加入個体が前衛3枠目へ配置される
- 加入処理の再読込で個体が増殖しない
- 全滅後に道具と遭遇状態が復元される
- 再調査では既知条件が表示される

## 14. 次に決めること

導入から灰苔湿原の分岐までの物語会話量を決める。短い会話と観察記録を中心にし、チュートリアル中の長い説明文を避ける方針を推奨候補とする。
