# 封域異獣録

『封域異獣録』（ふういきいじゅうろく）は、iPhoneのホーム画面から遊ぶ個人用PWAとして計画している、オリジナルの異獣調査・育成・継承型ターン制RPGです。

少し暗い生態研究ものを基調に、滅びた古代錬成文明の痕跡を組み合わせます。最初の完成目標は、異獣6種、調査地域1つ、地域ボス1体を含む30〜60分の縦切り版です。

## 現在の状態

企画・仕様設計中です。アプリのソースコードはまだ作成していません。

確定している基本構成は次のとおりです。

- iPhone向けPWA、Windowsで開発
- React＋TypeScript＋Vite
- 前衛3体＋控え3体のターン制戦闘
- 地点選択＋短い分岐調査
- 観察・鎮静・協力要請による仲間化
- 提供元が消失しない「共鳴継承」
- IndexedDBへの保存とJSONバックアップ
- 生態調査記録帳を基調とした2D表現
