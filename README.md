# 封域異獣録

『封域異獣録』（ふういきいじゅうろく）は、iPhoneのホーム画面から遊ぶ個人用PWAとして計画している、オリジナルの異獣調査・育成・継承型ターン制RPGです。

少し暗い生態研究ものを基調に、滅びた古代錬成文明の痕跡を組み合わせます。最初の完成目標は、異獣6種、調査地域1つ、地域ボス1体を含む30〜60分の縦切り版です。

## 現在の状態

企画・仕様設計とPWA保存基盤に続き、灰苔湿原の入口から地域ボス「ニゴリグイ」の鎮静、地域報告までを実装済みです。`app/` では、前衛3体による単体戦・複数対象戦、遺跡波動の原因を順に解く生態接触、3段階の環境ギミックを持つ非討伐ボス戦を遊べます。灰苔湿原は、先遣隊記録3/3・水路弁2/2・異獣録6/6まで一周できます。個体育成、共鳴継承、音声、iPhone実機検証は未完了です。

技術スパイクで確認できたこと：

- React＋TypeScript＋Viteによる画面基盤
- 研究所、編成、異獣録、設定のHashRouter遷移
- 純粋TypeScriptのコマンドによる状態更新
- Dexie経由のIndexedDB保存と再読み込み復元
- JSONバックアップの検査、書き出し、復元
- Manifest、Service Worker、オフライン用事前キャッシュ
- 390×844のiPhone相当表示での基本操作
- 地点イベント、初回観察戦、仲間化、呼称入力、分岐選択
- 戦闘途中の自動保存と再読込復帰
- 旧セーブのスキーマ版1〜版5→版6移行
- 前衛3体＋敵1体のHP、活性値、対象、優先度、再現可能な乱数
- 観測櫓跡、キリハネの控え加入、先遣隊第1記録、上流弁復旧
- 沈み水路の短い分岐調査、複数対象の汚染戦、先遣隊第2記録、下流弁復旧
- 濾過樹群の4段階生態接触、レキマトイの控え加入、遺跡触媒、中枢解放
- ニゴリグイの3段階調査戦、排出暴発の予告防御、浄化経路接続
- 地域報告3択、第3記録、異獣録6/6、灰苔湿原の地域完了
- 型検査、Lint、単体テスト49件、本番ビルド

iPhone相当表示では、新規セーブから観測櫓跡、沈み水路、濾過樹群、地域ボス、地域報告までの一周プレイテストを完了しました。最終成果は記録3/3・弁2/2・異獣録6/6です。iPhone実機でのホーム画面追加、機内モード再起動、音声は未確認です。次は実機確認と人の操作による所要時間評価を行い、その後に個体育成と非消失型の共鳴継承を実装します。

確定している基本構成は次のとおりです。

- iPhone向けPWA、Windowsで開発
- React＋TypeScript＋Vite
- 前衛最大3体＋控え最大3体のターン制戦闘
- 地点選択＋短い分岐調査
- 観察・鎮静・協力要請による仲間化
- 提供元が消失しない「共鳴継承」
- IndexedDBへの保存とJSONバックアップ
- 生態調査記録帳を基調とした2D表現

## アプリの起動

WindowsのPowerShellでは、環境によって `npm.ps1` が実行制限されるため `npm.cmd` を使用します。

```powershell
cd app
npm.cmd install
npm.cmd run dev
```

品質確認は次のコマンドで実行できます。

```powershell
npm.cmd run lint
npm.cmd run typecheck
npm.cmd test
npm.cmd run build
```
